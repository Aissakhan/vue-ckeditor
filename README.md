# vue-ckeditor

> Ckeditor directive using Vue.js for Vue 2.0

![](screenshot.png)

## Build setup

``` bash
# install dependencies
npm install

# server with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
